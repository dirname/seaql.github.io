# 什么是 ORM

对象关系映射器（ORM）是一个编程库，它帮助您从面向对象的编程（OOP）语言与关系型数据库进行交互。

数据库中的表和列被映射到对象和属性，而额外的方法允许您从数据库加载和存储数据。

用 Rust 构建的服务是轻量级的（小二进制文件大小，低内存使用），安全的（带有编译时保证），正确的（如果单元测试设计良好），以及快速的（编译时优化将运行时开销最小化）。

由于 Rust 是一种静态的、强类型的、编译的、线程安全的、非垃圾收集的、非常规的面向对象语言，在 Rust 中使用 ORM 与您已经熟悉的其他脚本语言有所不同。

SeaORM 试图在避免 Rust 编程时的小问题的同时，帮助您获得上述优势。
