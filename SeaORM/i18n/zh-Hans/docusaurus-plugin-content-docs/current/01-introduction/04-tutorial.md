# 教程与示例

如果你更喜欢逐步的教程来学习如何使用 SeaORM，你可以查阅我们的官方 [SeaORM 教程](https://www.sea-ql.org/sea-orm-tutorial/)。还有一些由 [社区](https://github.com/SeaQL/sea-orm/blob/master/COMMUNITY.md#learning-resources) 编写的优秀教程！

你也可以查看 [SeaORM 烹饪书](https://www.sea-ql.org/sea-orm-cookbook/)，以获取一些常见问题和使用 SeaORM 的推荐实践。
