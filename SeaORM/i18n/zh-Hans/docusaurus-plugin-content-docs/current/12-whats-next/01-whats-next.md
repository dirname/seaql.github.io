# SeaORM 的下一步是什么？

我们仍然记得两年前首次向 Rust 社区[介绍 SeaORM](https://www.sea-ql.org/blog/2021-09-20-introducing-sea-orm/)的时刻。

如今，许多开源项目、初创企业和公司都在使用 SeaORM。如果你的公司正在使用 SeaORM 并希望被展示，请[提交你的评价](https://forms.office.com/r/YbeqfTAgkJ)。

如果你在使用 SeaORM，请参与[社区调查](https://www.sea-ql.org/community-survey)。你的反馈将影响我们对项目未来的决策。具体来说，当前的功能集是否足够被认为“稳定”？我们应该将此版本命名为`1.0`吗？或者我们应该将其标记为`LTS`并长期支持它？这会促进采用还是阻碍进展？

开源项目是一项永无止境的工作，我们正在积极寻找维持项目的方法。SeaQL.org 是由热情开发者运营的独立开源组织。你可以通过给我们的仓库加星和分享以及通过[GitHub 赞助](https://github.com/sponsors/SeaQL)捐款来支持我们。

我们正在考虑多个方向来为组织创收。无论我们做什么，我们都想保持非营利组织的身份。如果你有任何建议，或想加入我们或与我们合作，请通过`hello[at]sea-ql.org`告诉我们。

感谢你迄今为止的支持，让我们共同努力，使开源可持续发展。
