# 教程与示例

如果你更喜欢逐步教程来学习如何使用 SeaORM，你可以查看我们的官方 [SeaORM 教程](https://www.sea-ql.org/sea-orm-tutorial/)。也有一些很好的教程由 [社区](https://github.com/SeaQL/sea-orm/blob/master/COMMUNITY.md#learning-resources) 编写！

你也可以查看 [SeaORM 烹饪书](https://www.sea-ql.org/sea-orm-cookbook/) 来获取一些常见问题和使用 SeaORM 的推荐实践。

如果你非常急切，想要立即上手的东西，SeaQL 维护了一系列由社区贡献的官方示例（我们欢迎更多！）：

+ [Actix 示例](https://github.com/SeaQL/sea-orm/tree/master/examples/actix_example)

+ [Poem 示例](https://github.com/SeaQL/sea-orm/tree/master/examples/poem_example)
+ [Rocket 示例](https://github.com/SeaQL/sea-orm/tree/master/examples/rocket_example)
