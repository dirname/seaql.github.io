数据库连接

要获取数据库连接，请使用 Database 接口：

let db: DatabaseConnection = Database::connect("protocol://username:password@host/database").await?;

protocol 可以是 mssql: 等等。

host 通常是 localhost、域名或 IP 地址。

tip

如果无法使用 localhost，尝试输入 IP 地址和端口号，例如 127.0.0.1:1433 或甚至 192.168.x.x。

在底层，会创建一个 sqlz::Pool，并由 DatabaseConnection 拥有。

每次调用 execute 或 query_one/all 时，将从池中获取并释放连接。

多个查询会并行执行，您可以使用 await 来等待它们完成。

连接字符串​

以下是一些数据库特定选项的提示：

MSSQL​

指定架构​

如果架构是 dbo，可以简单地写：

mssql://username:password@host/database

或者，通过提供额外的 currentSchema 查询参数来指定架构名称。

mssql://username:password@host/database?currentSchema=my_schema

信任对等证书​

您可以通过提供额外的 trustCertificate 查询参数来信任对等证书。

mssql://username:password@host/database?trustCertificate=true

连接选项​

要配置连接，请使用 ConnectOptions 接口：

let mut opt = ConnectOptions::new("protocol://username:password@host/database");
opt.max_connections(100)
    .min_connections(5)
    .connect_timeout(Duration::from_secs(8))
    .acquire_timeout(Duration::from_secs(8))
    .idle_timeout(Duration::from_secs(8))
    .max_lifetime(Duration::from_secs(8))
    .sqlx_logging(true)
    .sqlx_logging_level(log::LevelFilter::Info);

let db = Database::connect(opt).await?;

检查连接是否有效​

检查与数据库的连接是否仍然有效。

|db: DatabaseConnection| {
    assert!(db.ping().await.is_ok());
    db.clone().close().await;
    assert!(matches!(db.ping().await, Err(DbErr::ConnectionAcquire)));
}

关闭连接​

连接在被丢弃时将自动关闭。要显式关闭连接，请调用 close 方法。

let db = Database::connect(url).await?;

// 在这里关闭连接
db.close().await?;
