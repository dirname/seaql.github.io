实习 @ SeaQL

January 28, 2023 · 5 min read
SeaQL 团队
Chris Tsang

SeaQL.org 提供针对大学生的实习机会。实际上，这将是 2023 年的第 3 届实习生。

实习通常在夏季和冬季学期的假期进行。在实习期间，您将专注于一个项目，并在结束时发布该项目的成果。

我们运营模式的显著特点是涵盖软件开发的整个生命周期，从设计 ➡️ 实现 ➡️ 测试 ➡️ 交付。您将会惊讶于在如此短的时间内能取得多少成就！

迄今为止，StarfishQL ✴️ 和 Seaography 🧭 是我们团队在过去一年中创建的杰出项目。我们以精心规划、一致执行和务实的方法为荣，我投入了大量时间进行想法评估：如果项目范围太小，将会没有趣味，而如果范围太大，则将无法按时交付。

亲爱的本科生们，这里有几个您应该参加像 SeaQL 这样的开源组织实习的好理由：

  1. 明确的简历展示：开源项目是公开、可检查的，并具有现实世界的影响力。我们还确保它具有良好的品牌形象、图形设计和可见度。
  2. 不受商业流程驱动，我们不妥协于工作质量。我们没有专有的开发流程，因此所有使用的工具都是开源的，具备可转移的技能。
  3. 您将为社区作出贡献，并与来自世界各地的人进行互动。开源协作是人类所发明的最佳事物。您只有亲身体验后才会相信我的话。
  4. 因为您是所工作项目的推动者，这让您能够更多地发现自己，特别是能力和自律：在某个方面，您总会低估或高估自己。

以下是您将学习的几个内容：

  1. “思考 > 编程”：您花在前期思考上的时间越多，写出的代码就会越好。您花在后期审查上的时间越多，代码也会越好。
  2. 如何组织代码库。充分利用 Rust 的类型系统，构建模块化、可测试的代码库。
  3. 测试自动化。每个 SeaQL 项目都经过持续测试，这是我们工程过程的重要组成部分。
  4. 文档。在文档编写上，我们的软件旨在提供全面、易于理解且有趣的教程。
  5. 性能调优。根据项目的不同，我们可能会进行一些基准测试和优化。但一般来说，我们会让您避免编写产生不必要开销的代码。

我们在 GSoC 2022 中是一个指导组织，可能会在 2023 年再次参与（更新：我们未被 GSoC 2023 接纳）。我们还提供 GSoC 以外的实习机会。那么，成为贡献者的要求是什么？

  1. 保持热情。您必须表现出对开源和软件工程的热情，因此需要有一个良好的 GitHub 个人资料，并有一些参与记录。
  2. 全力以赴。这是一份全职工作。虽然工作完全远程且时间灵活，但您在规定的实习期间内不得有其他承诺或工作。
  3. 保持开放的心态。您应该认真倾听您的导师，并根据他们的建议采取行动。
  4. 多写。以有组织的方式在所有渠道上交流您的想法和进展。

不过，您不仅要听我的话。以下是我们过去实习生的心声：

  • Sanford Pun - 在创建 StarfishQL ✴️ 的团队中：在 Vision Cortex 和 SeaQL 实习
  • Panagiotis Karatakis - 在创建 Seaography 🧭 的团队中：我的 GSoC 经验

为您即将到来的技术行业职业生涯做好充分准备！现在就关注我们的 GitHub 和 Twitter，并留意未来的公告。
