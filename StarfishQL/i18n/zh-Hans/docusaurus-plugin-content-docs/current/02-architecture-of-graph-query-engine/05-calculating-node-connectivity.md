# 计算节点连通性

在最简单的情况下，可以通过检查每个节点的入连接和出连接数量来处理节点连通性。图论中已有的度量标准包括有向图中的入度和出度，以及无向图中的度。

然而，这些现有的度量标准仅提供了有限的图形分析能力，因为它们只考虑了1层的图遍历，忽略了图的递归性质。因此，我们设计了权重衰减连通性度量标准，以支持更深入的图分析。

如果我们定义权重衰减因子 `f` 为一个在[0, 1]范围内的标量，我们可以通过递归采样其祖先/后代节点的度数来获得一个节点的连通性，如下所示：

不失一般性，假设我们想要找到节点 `n` 的 *入连通性*。

```rust
find_in_conn(n, f):
    如果 f 足够接近 0.0:
        返回 in_deg(n)

    令 in_conn = in_deg(n)
    对于 n 的每个子节点 c:
        in_conn += f * find_in_conn(c, f * f)
    返回 in_conn
```

*注：这个算法可以使用堆栈/队列代替递归来提高性能。此外，在这个简单的算法中忽略了包含循环的图，但在实践中处理起来很简单。*

<a id="weight-decay-factors"></a>

不同的权重衰减因子会产生不同有趣的连通性度量标准，最值得注意的是：

- 当它是 `0`（*简单连通性*）时，连通性仅仅是平凡的度。
  - 在这种情况下，由于权重实际上立即衰减到0，因此 *衰减模式* 是 *即时*。
- 当它是 `1`（*复合连通性*）时，连通性会考虑到感兴趣节点的连通组件中的**所有**祖先/后代节点。
  - 在这种情况下，由于权重实际上从不衰减，因此 *衰减模式* 是 *无*。
- 当它介于两者之间（*复杂连通性*）时，连通性在整个遍历过程中*衰减*，对于某些应用场景来说这是一个很好的属性。
  - 在这种情况下，*衰减模式*与速度相关：*因子越小，衰减速率越快*。
